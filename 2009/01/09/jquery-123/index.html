  • 一个挺好玩的叫做pageSlide的插件
  • Shawn 的一篇文章，给你的 WP 弄个进度条。

pageSlide 粗看没啥特别，细看应该也就是这样而已…… 不过重要的是这个想法，而非技术。 以前看到有篇说 jQuery 的文章就指出，虽然 JavaScript 弄出来的许多特效都很不错， 然而存在着很大误区就是其实对页面加载速度毫无帮助。比如点击“查看更多”显示文章的剩余部分， 其实只不过是将页面上的隐藏元素显示出来而已。真正的简介高效在这里应该是先载入一部分， 剩余的再用 ajax 根据用户需要获取。我个人其实也支持这种观点，但是我经常会变得非常纯粹主意，所以您看着办。

进度条也是如此，虽然没啥特别之处，进度也都是人肉设定，但是对翔仔挂在嘴边的“用户体验”来说，确实大有裨益。
