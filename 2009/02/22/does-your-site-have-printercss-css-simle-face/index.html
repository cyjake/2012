你的网站有 printer.css 吗？外一则，CSS 笑脸。

Feb 22nd, 2009 | Comments

当然，如果网站正文其实不多，自觉不会有人傻到想打印的话，可以无视。

当然，你也可以写在同一个样式文件里头：

@media print { // css print rules here }

新闻、维基、文档类的网站一般都还是需要的，GNU 的文档除外，丫根本一点样式都木。样式其实也简单，把侧边栏和页首的导航栏去去掉，把其他同样是功能性的区块也去去掉也就是了。不过有人比较无聊：

window.onbeforeprint = function() alert(“You’re printer is out of paper!”); }

我的博客文章通常都短，对生活影响不大，对广大劳动人民的指导意义也不多，基本没有必要。所以就无视了。
