轮滑联赛？

Feb 8th, 2010 | Comments

今天看了部电影，《Whip It》，名女人 Drew Barrymore 和 Ellen Page，话题是轮滑。

没想到真的有 Roller derby 这么一个体育运动，而且历史悠久，est. 1922。Drew 同学拍电影眼光真的很不错。当然，我看过的就两部，样本容量低了点。执行制片并投资的《Donnie Darko》（我最爱的电影）和执行制片并执导筒的《Whip It》。前者是对初出茅庐的电影学院毕业生的帮助，后者算是对女性主义的赞同？而电影的情节副线，Baby Ruthless 的选美（在妈妈的安排下参与，最终放弃），正是女性主义对传统女性的不喜吧？

Rollers Derby 运动本身，就极具女性主义色彩。当然，它本身是男女皆可的。但是往往是满场短裙丝袜。而且就这个运动的规则，给男筒子去玩，恐怕护具还得加厚一圈。电影里头，Drew 演的 Smashley Simpson，骑在未婚夫背上玩骑驴，算不算“今儿个姐儿做主”的架势？

呃，算了，我不擅长扯这个。

Ellen Page 呢，则一如既往给电影带来了很赞的原声。当然，这不能算在她头上。只是在我看过的两部她演的电影里头，《Juno》和本片，原声都超级赞…… 再一次，样本容量低了一点。

电影最赞的是泳池欢好那场戏…… 连我这个不会游泳，人生到目前为止没去过几次泳池的人，都看的蠢蠢欲动。呃，或者说我看了任何那个啥的镜头都蠢蠢欲动？恩，有许多很让人冲动的场景的，诸位不妨去看看“好想侵犯楼主啊”系列（十八岁以下请勿模仿，当事人觉得不舒服请勿强迫……）。

最后说点体育沾边的，前一阵子我也买过几个毽球踢。顺便去网上看了些毽球比赛的视频。不温不火…… 也曾yy过，是不是可以换种比赛方式，来点身体接触什么的。像篮球那样弄俩篮子，像蹴鞠那样弄几个门洞，像 Quiditch 那样弄几个圈，比这种抬腿杀球会有意思一点吧？

Landon Pigg （好好玩的姓）同学的歌还是不错滴，人长得有几分像 In Brudges 的 Chloe、Harry Potter And the Goblet of Fire 里的 Fleur（没错，同一个人，Clemence Poesy） 哈哈。

夜了，睡觉。

Ant 啊 Ant

Feb 8th, 2010 | Comments

快过年了，折腾个人历史遗留代码。其中有个学 jQuery 的时候弄的工作日志记录工具，JavaScript 部分写得不咋地；Java 部分写得更囧。于是打算给起个新名字，好好折腾一番，重点放在传说中的 Plug and PrayPlay 上，看能否得众 PM 芳心，拿来部署到项目组内部作为 Recreation Affair 的告示板、讨论区和记账本。

芳名 Aragog。

啊，离题了。因为要脱胎换骨，所以在代码组织方式上有所变更。因为蛋疼，自己写 ant 的 build.xml。在删除部署目录的时候，一开始还想稳妥起见，写成这样：

<delete> <!-- 漏掉了 dir 参数，应该是 delete dir="${deploy.dir}" -->

于是项目目录下的 blah 文件夹里的东西全没了。

魅族 M8 二三事

Feb 7th, 2010 | Comments

今天去魅族官方网站，发现更新了新固件，0.9.6.9，新界面，很唬。加上 1 月份破茧成蝶的 QQ for M8，M8 已经绝对达到可用的标准了。而我用了 M8 将一年多，磕磕绊绊，最近赶巧郁闷到一块去了，就拿出来说一下。

差不多两个星期前，我的 M8 被我摔了一下，从桌上掉到脚盆里，脚盆裂了，手机当时没事。心说魅族牛鼻啊，质量上升到诺基亚一个档次了。后来陆陆续续的发现触屏不准了，触点歪了等等，还一度以为是新固件的问题。前些天直接罢工，怎么摸都没反应，死鱼一样。于是发现我杯具了。

为了回收旧机（以前提前放出的测试机），魅族推出过换新的活动，按折损贴现换取新机，09 年圣诞节的时候已经停止了。我的是测试机，二奶机黑莓 7290 老早甩掉，正室又跟我感情破裂。原本包娶包换的魅族婚介公司，又不让换了，于是我就只能拿去修，结果专营店的人说，年末了厂家不接货，等来年吧……

总结经验教训就是：触屏手机质量还是不够威武，我的第一部手机索爱T628，方向键会有点小问题，但也一直可以坚持用，这触屏一挂全都玩完，只剩个接电话，其一；要尝新，比如说用测试机，HTC G1，要有一颗会折腾并抗打击的心，我二者皆欠奉，杯具了之后就打算换手机了，其二；魅族的东西还是不错的，手机做得也用心，软件开发方面可能初涉，做的不好的地方也不鲜见，要有一颗包容的心，M8 还是有前途的，其三。

我的话完了。

最近二三事：台球、驾照、想家

Jan 27th, 2010 | Comments

博客搬家之后，知道地址变更的寥寥无几；不过本来也没多少个，也没啥好伤心。好像很久没有交待最近做点啥，今晚又打台球打得郁闷，只好到这里写博客。

从 2009 年年底到 2010 年初，我去打了挺多次台球，大多数时间被虐。但是一有机会还是会去，还曾一度考虑将来买张台球桌子。一问老板价钱就却步了，9800 块大元。

今天晚上又和俩同事去打。三个人轮替十分不爽，手感全无。吸了很久二手烟，喉咙昨日嗑瓜子磕到太狠发炎，整个人就彻底萎靡了。慎重考虑一下，要改正生活和饮食习惯。

其实，一般表面上把积极向上挂在嘴边的人，内心都很抑郁。海子同学说：

从明天起，做一个幸福的人 喂马，劈柴，周游世界 从明天起，关心粮食和蔬菜 我有一所房子，面朝大海，春暖花开

结果他卧轨了。

春节将至，我无限想家。有人说，消极往往跟消沉同义，情绪低落往往很难爬出来，即所谓 pessimism indulgent。很久以前，我翻译过一篇《焦虑、消极、程序和我》，讲的大多都是这些事情。大家摆脱情绪低落的办法都不一样，喝酒、看片、血拼、打扫、锻炼身体等等。

我还没找到适合自己的。或许熬个两周，回家了，就好了。

哦对，还有其他一些麻烦事。与同事合租的房子快要到期了，三月份。其中一个室友铁定要搬走了，估计是跟我和另一同事俩大老爷们合租不太爽。另一同事也琢磨着是不是要搬，尚未确切。所以，过个年之后回来，可能又得搬家。还没想好这次该怎么办。

还有就是学车，杭州市驾驶培训报名处的俩检查色觉的医生之一，已经认得我了。从08年年底到现在我总共跑去试着报名了三次，都被卡住了，屡试不爽。我看得见红绿灯、色弱；我需要学车；我公司离家很远但是有车可以上绕城高速 20 分钟；我为啥买得离家这么尴尬因为市区房子太贵；我每次去的经历都不愉快因为事业机关的工作人员态度都很烂；我求情摆事实讲道理都无济于事；我很拧巴。

解决办法不是没有，去余杭区临平镇、回温州学、外省买证等等。我就是很奇怪为何我需要怎么拧巴。

我看得见红绿灯，我看不清他妈的红绿小点点，我没资格开车。

好想回家。

综上所述，我蛋疼了。

Flex 与 ElementTree 里头 XML 的 Namespace 的问题

Jan 23rd, 2010 | Comments

Flex、ETree、XML 的 Namespace…… 好奇怪居然可以扯到一起。

Adobe® Flex 是个对自家脚本语言 ActionScript 的封装，弄了许多趁手的控件、Layout 工具。ActionScript 因为有 e4x，可以说是完美支持 XML。前些日子试着在 Flex 里头操作一个既有的 XML，准确的说，是个 Jasper Report，.jrxml 文件。导入之后 e4x 的趁手的 dot(.) 和 attribute(@) 全然不起作用。后来发现是 XML 的 namespace 的问题。

解决办法简单，把该 XML 的 namespace 定义一下然后 use，就可以了。

var foo = "http://www.adobe.com/flash/proxy";
use namespace foo;

因为如果不指定的话，e4x 去找的 tag 和 attribute 确实 namespace 前缀，就找不到了。

而后来在 ElementTree 里头碰到的问题也是一样。ETree 是 Python 2.5 自带的除了 dom 与 sax 以外的另一个 xml 操作库，写起来要简洁一些，只是木有 e4x 方便。同样，因为 XML 的命名空间的关系，使用 ETree 的 .find 的时候，如果忘了加命名空间前缀，就会找不到，所以应该这样写：

namespace="http://xspf.org/ns/0/"
tree.findall('.//{%s}track' % namespace)

麻烦的是，ETree 里暂时木有可以 once and for all 指定 namespace 的办法。

顺带一提，Python 里头函数调用时，如果有参数叫做 class，直接显式地写实不行滴，解决办法是：

w.element("Node", **{'class': 'oops'})
