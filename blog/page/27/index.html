Apache Friends: XAMPP, and One Trouble Shoot

Apr 16th, 2009 | Comments

xampp 有点像以前同学拷给我的某某大学 LAMP 套装之类，也是个对 Apache、MySQL、PHP 等服务的集成，界面清洁、操作简易，让我很是心动。

装了之后 Apache 跑不起来，原以为是跟 McAfee 冲突，Google apache mcafee conflict 的话倒也不是空穴来风，有个 slimftp 跟 mcafee 确实冲突。只是我的是最基本的 httpd 都动不起来。咨询室友，瞬间帮我搞掂。我也不晓得是个啥子道理，enable LMHOSTS lookup 就行了。

IBM Rational Functional Tester

Apr 10th, 2009 | Comments

Environments 设定完全没问题的，IE 测试就是通不过，错误信息如下。

HtmlTestDomainImplementation Loaded = false HtmlTestDomainImplementation Name = Unknown

Google 之后在 IBM 的官方论坛找到了解决办法，控制面板 -> Java -> Advanced Tab -> Java Plug-in -> Enable the next generation Java Plug-in 不要勾选。

重启下浏览器，就可以了。

“以用户为主导的设计”

Apr 5th, 2009 | Comments

最近看到白鸦跟 fenng 掐起来了，UCDChina 在宽屏高分辨率下就傻逼了。证据如下：

看了下笔仗，白鸦的论据无非固定宽度用户的眼睛不用因为左右移动幅度太大而太累，Google 等都是固定宽度， 大家的分辨率普遍都是 1024 之类。

我的看法是丫的题头原话奉还，“你太土了，我得给你好好上上课”。

  • 既然是个大型公共网站，域名都牵涉到 China 了，就不要在自以为是地认为大家的显示器都是 1024px 宽。 4k 价位的台式机都能 20” 液晶了你有什么依据认为大家活在旧社会？
  • 既然想固定宽度让用户眼睛少动一点，那就行行好顺便把整体的宽度也固定一下。侧边栏是没正文重要， 但用户还是要点的。它是你的网站导航、功能索引、工具栏等等所在的地方。24” 宽显的用户鼠标移起来会很累诶。 你不会说用户既然是大显就不要最大化浏览器窗口了吧？你自己说的“用户中心”哦，别说一套做一套。
  • google、baidu 的搜索结果没有侧边栏，只有广告位。别把网站功能与内容弄得天各一方。 另外，google 的绝大部分设计都是经历过用户测试的，他们可以在数十种颜色方案之间摇摆， 可以24小时内获取大量用户体验数据。UCDChina？反正我不上，在它够牛掰之前， 还是把网站基本设计搞搞好，别盯着所谓的 1024 不放。

既然只是做设计的，就请些牛掰的前段开发者给你弄网站，别分辨率一变就土鳖了。

最近二三事

Apr 3rd, 2009 | Comments

4月1日的草稿，其实也就是一个标题。今天翻出来，又因为最近更新得比较少，就觉得还是写点东西的好。

吉他还是有在学，小海老师教了手指扫弦的特殊技巧，和布鲁斯入门。扫弦和和弦切换一直是我的大弱项，估计一时半会也不会有啥改进，加上唱歌一直非我所长，也就不怎么上心。而布鲁斯，则大大勾起了我的兴趣，摇摆的感觉很好玩，期待后续的曲子。《指弹吉他完整教程》最近卡住了，有两周没进展了估计，Scarborough Fair 的 B 级难度我就连不上去了，还得好好研究视频。A 的话，倒是轻轻松松，就是左手手指老是疼。

公司最近很闲，米国那边管理层地震，都无暇估计 TC 这边。许多项目都停摆，不知还有否可能重见天日。我练手 jQuery 与 Ajax 的小项目一直未得组里同事的正经对待，在尝试了几次成功或失败的破坏之后，就不再访问了。所以，数据库里的数据至今寥寥。数据的基本统计，和数据挖掘之类的工作，因为数据量比较尴尬 ，其一；自身技术比较弱，其二；也就未尝展开。

画画倒是一直又在继续，大部分时间依然停留在画脑袋的程度，画了自画像，画了我姑娘的背影，没拍zp，回头发上来给诸位看看。她的头发真好看。

还比较大的事情就是我昨天晚上连夜坐汽车回家了，19:40 到温州，再打的回家，到家已经次日 1:00 了。我妈因此大为光火，“太孩子气了”。回到家，工资卡里剩50，钱包剩 50，还问老爸要了 100 帮我付打的钱，加上之前欠同学 500，同事 300，这就是我目前在杭州混的状况，囧。

哈尔的移动城堡和宫崎骏

Apr 3rd, 2009 | Comments

标题起得很装13，点进来发现是水文一篇的筒子勿怪……

前些天从姑娘那里拷来她从缘网下的宫崎骏全集，发现没有千与千寻，答，那部我已经看过了就没下…… 囧。于是看了《哈尔的移动城堡》。

看的时候注意力很分散。画师们的画工，苏菲点亮帽子店客厅的煤油灯的时候背景色彩的变化；配音配乐的出彩，马路、人群，苏菲坐电车时候的背景乐；片片断断听出一二的日语；老奶奶苏菲的吐槽，例如“ite，看来老人还是一点都不好当”，让菜头给找个住处最好的“看来人老了确实会变坏呢”之类。

所以于故事本身倒也不怎么在意。倒是豆瓣上的讨论页面让我眼前一亮，有个话题说“哈尔一定是天平座的，‘不美了，还不如去死……’”。天秤座真的有这种尽善尽美的心态？

练完《灌篮高手》之后可以考虑买本宫崎骏电影原画集来临摹看看。
