最近二三事

Dec 12th, 2010 | Comments

年末了，产品经理要在项目发布截止之前改 feature，于是要加班。楼上的开发们都很努力，周末整天都在，下班比平时还晚。我痛恨这种所谓紧凑的项目计划，扭捏着不愿意去，但是不去又会拖住开发的进度，只好过去。

周二周三去了一趟北京，参加 O’Reilly Velocity 大会。淘宝网是主办方之一。北京的冬天很舒服，屋里头有暖气，很给力。遗憾就是空气太干燥了些。大会主题是 Web 性能与运维大会。有做运维的讲可用性，6个9神马的；也有做前端的讲各种优化技术。很给力的是 Facebook 的蒋长浩童鞋，分享了优化 Facebook 的三项技术，Ajax 化的 Quickling，更进一步的 PageCache，再上一层楼的 BigPipe。也讲到一些细节，十分吸引人。怪不得诸多技术大拿喜欢去 Facebook。它的前端后端，确实都很给力。难能可贵的是都有华人的身影（还有做 PHP 2 C++ 的 Hiphop 程序的赵海平）。本公司也有大牛分享，同样不错。此外也有些公司来做广告。比如 IE9，比如 chrome。

首都国际机场 3 号航站楼模样十分给力。

晚上和女朋友一起去吃哈里欧咖啡的牛排神马的。她在快抱网团了个十分给力的套餐，我木有参加，则全价作陪。T 骨牛排非常嫩，很好吃。咖喱牛排饭也不错，可惜牛排肉老了一点。这家的西餐做得比万塘路靠近物美那家皮萨店是绝对要地道多了…… 神马时候我楼下也能有这样的店，杭州就是个大城市了。

最近二三事

Dec 5th, 2010 | Comments

到了新公司之后博客写得少了，工作上比较忙，处在应付各种需求各种产品经理与交互设计师的奇思妙想的阶段。理想中的项目组是彼此对技术的理解没有范围限定，了解对方的手艺方便的与不方便的地方。实际情况呢，则想去很远。比如一个页面上的类似当年 Gmail 容量增长的计数器效果，我会被要求只去负责该样式，而具体实现让做 Java 的人负责（WTF……）。

周二周三要去要去一趟北京。从小到大很少出省，前一阵跑了两次上海，看了几眼魔都，觉得很不错啊。今番要去宇宙中心，心中有些忐忑。

我在公司的项目（前端部分），比较乱，YUI、jQuery 和组里元老的老 js 框架，代码写得有些奇怪。整个项目可以优化的部分有很多，然而开发周期往往不够长，一两周发布一次，有心杀贼无力磨刀了都。而短周期的最直接害处就是，容易让你放弃礼仪廉耻，忘记你的代码终究有一天会被人看，被人改，怎么快怎么来。工作完成的判断标准就是“it compiles！”或者“it works！”

不过这样的环境对我的好处是，可以少点拖延…… 不再能够有时间去想我这样写是不是太笨了点。

外公已经出院了，回老家养病，地里还有他惦记着的番薯。小舅舅跟我说，外公外孙这么多，最惦记的还是我，感动得我稀里哗啦的。我们生在这个世界上，最期盼的莫过于被重视，平素里最喜欢标榜的也是自己的个性，认为人人尤其独特处。网络让信息铺天盖地地倾覆到我们脸上，告诉我们，绝大部分时候我们只是统计数字。所以做了前公司两年统计数字之后，我选择离开了。我不在隶属的公司工作，而是被外派到一家外企。离职的时候，外企的项目组老板跟我说，随时欢迎你回来。

和女朋友快两年了，我觉得自己思不够敏，言是足够讷的。很多时候都喜欢把东西写下来解决，时代变得太快，小女朋友不习惯这个，可我有些话还是习惯憋着，受不了了，胡乱发泄一番，对彼此都不好。

新公司有个同事，逐飞，弹吉他的。他和新公司的一些同好，组了个乐队。上周他买了吧电吉他，看着十分给力。我虽然久已不弹，心中还是喜欢的。他说乐队少个贝斯。我犹豫着要不要买把练练。

驾照考出来两个多月过去了。国庆节的时候开得很爽，之后没摸过方向盘。换工作很受伤，新工资依然养不起车，犹豫到底该不该买。

差不多就是这些了。

Ie6 Z-index 失效 Bug

Dec 5th, 2010 | Comments

Web 2.0 的一大特征就是页面不再是平的，我们开始有各种浮层、窗口、提示，骚一点的，还会给页面蒙上一层阴影，半透明的哦。这些浮层通常都很开心地像浮云一般飘在那边，只要它们没有遭遇 ie6。

浮层盖不住页面的原因有很多，其一是 ie6 中，select 控件是以 Window Shell ListBox 和 ComboBox 方式实现的，不支持 z-index、title、zoom 等，层级很高，不与 div、span 为伍，超脱于 z-index 之外自然就是世外高人了。解决方式就是用一个同样牛逼但不会无视 z-index 的 iframe 来把丫盖掉。公司里不鼓励用 jQuery，这在我看来很是无厘头，可能跟程序员基数有关系，只是未免不够拥抱变化。所以 bgiframe 插件就不好用了。不过不用也好，毕竟为 ie6 添加个 workaround 就得加个 js 请求，有点夸张的。贴一个 YUI3 写的示例。

另一个 ie6 的浮层 bug，则是 z-index 失效，而该控件明明是支持的。导致这个 bug 的原因有二：浮层或者浮层的子节点被 float 了，其一；父节点有 position: relative; 其二。两个因素都很屎。解决办法相应的就是清除 float；为父节点设 z-index。

IE6 Overflow:visible Bug

Dec 5th, 2010 | Comments

ie6 甚至 ie7 让前端开发抓狂的原因在于，这俩货常有些奇奇怪怪的地方，而这些问题通常是都可以以很诡异的方法解决的，代价就是时间。今天遇到的问题是，当给一个 #block 节点的 overflow 为 visible，并限制其高度为 200px，而它的子节点的高度有可能超出的话，ie6 会把该子节点的高度作为 #block 节点的高度。其他浏览器（firefox、ie7+、webkit）都没有这个问题。quirksmode 对此问题也有记录。ppk 比较乐观，说这个 bug 可以作为 ie6 的 min-height 的猥琐实现（是的，ie6 这货不支持 min-height、min-width、max-height、max-width）。

碰到这问题的时候想死的心都有了，求助 Google 没想到还真有解决方法。就是在 ie6 中，#block 的 overflow 设为 hidden，子节点 position: relative。

这货不是浏览器，这货是我上辈子的夙敌。

新工作

Oct 20th, 2010 | Comments

10月11日我正式到淘宝报到，并起了花名叫做逸才。次日正好百淘70期，师兄就让我先去参加这个，工作的事情再说不迟。百淘的全称是百年淘宝。除此之外还有百年阿里、百年支付宝之类。王小波说玩缩写近乎愚昧，是盲目的集体感的催化剂。因此参加这个培训伊始，我是以很谨慎的态度潜伏的。

培训首日讲了一些规矩什么的，为期七天，全脱产，不用管工作，不许迟到早退，务必尽量少请假，等等。来了几个老员工，将一些阿里巴巴、淘宝的往事。我可能不好讲得具体，不过应该也都人所共知。总之就是革命前辈的光辉故事。

次日野外拓展，因为下雨，不用玩一些高难度的事情例如高空抓杆什么的。一些小游戏，要求分成四个组的各组合作取胜。结束之后，培训师还会讲因势利导讲出各种大道理来。很有些大人哄小孩的调调……队友轻声跟我说，做开发的都是实在人，玩这培训真是没来由。

后续几日，都是些江湖前辈们来讲淘宝的架构、各种八卦，很不错。周六这天，刚好是重阳节。马云是个很知道迎合大众的人。不管是真情假意，淘宝网和淘宝的员工，确实也都在做一些公益的事情。而我们这些百淘学生，则要在重阳节这天出去采访老人。

这周的星期一星期二，是百淘的最后两天。星期一负责检查作业。江湖行采访（采访三个老员工，一年香、三年陈、五年醇），百淘个人感想等等。星期二则是倒立。铁木真说，倒立的目的不只是锻炼身体，而是让你知道你认为不可能的事情，其实也没那么大不了。马云则说，要换个角度看世界。马云的话很好听，铁木真的则很实际。

倒立之后，是各种表演，各种猥琐。过程非常精彩，各显神通。诸位平时看着道貌岸然的家伙平静的外表下，果然有颗风骚的心。

今天是我正式做这份工作的第一天。工作比想象的要繁杂。本以为很小一件事情，往往要沟通好几个人。难怪会有产品经理这样的工作…… 配置好了 emacs、firefox 和 putty（偶尔翻墙用），装了 svn、chrome 等。算是正式开搞了。

长假收尾

Oct 7th, 2010 | Comments

今天是十一长假的最后一天，同学们陆陆续续出门，返校返工。我因为新近离职，成了无业者，8、9号就无事可做，又因为家中还有些事可以帮上忙，便打算延宕几日。

早上出去买内存的时候碰到了发小儿陈献。他去数码城光顾战友生意，买些路由器什么的。这厮新跟人合伙开了家小店，卖饭团，当了老板。味道很不错。晚上应他之邀，去了万松路附近某处烤鱼店，又有幸邀到了远亲+发小阿黑先生。相顾之下，熟悉又陌生。

下午老鸭驾车返杭，原本有幸搭车，因为前面说的这些私事，只好爽约。载着妈妈往农贸市场采购水果的时候，又遇到了金老三同学。金帆同学新婚燕尔，发福不少。

晚上与陈献、阿黑聊，说到老家的各种事情，得知一个颇让人惊讶的消息。鹿木乡的垃圾中转站，要建在我们村，村里人因为得以新修其他基础建设，加上有五毛游说，表示对生活影响不大。垃圾中转站的选址在我们看来，充满了各种猫腻、厚黑，和各种奇奇怪怪的仇怨。我爱着这片土地，却吃不消这莫名其妙的政治。只能默默生气。

昨日自驾游

Oct 5th, 2010 | Comments

昨天跟着表哥的车，开到平阳西湾，顺便在新建的瑞安大桥上兜个来回。桥还是很气派的，就是超车道上就些人力三轮，十分不给力。新路大多在建，所以后来走了老路，路况很差，很多运沙车。开的车是小姨夫当年的进口尼桑，自动挡，老没牙了。车子比较宽，刹车和油门感觉倒是依然不错，但是加速的时候发动机偶尔会有很奇怪的声音。

温州这一块我全不熟，比较了解的只是瑞安市区，湖岭镇的老商业街和自己的老家鹿木乡。昨天跟车也不算好，表哥中间停着等了我很多次。后来返程的时候要直接去小姨家，就走的另外一条路。下了 104 国道之后，车子更加难开。那一块的路基本都在修，全是灰，天色又暗。我忙手忙脚，终于把示廓灯和大灯开起来了。

会车的时候，被远光灯晃得很惨。

自驾游的目的地是西湾，因为路程很短，目的地很迷你，这段为时数小时的旅行要称自驾游是十分勉强的。最大的收获是看到东海。因为离飞云江入海口比较近，海水很脏。玩的人不少，大多在拍照。小孩子们买了铲子、小车，在沙滩上筑堡。
