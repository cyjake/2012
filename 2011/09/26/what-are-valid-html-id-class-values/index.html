HTML Id、class 属性值

Sep 26th, 2011 | Comments

在还是草稿的 HTML5 规范里头，id 和 class 属性的值的约束似乎没怎么说明，只说 id 不能包含空格。我们这里讨论，还是以 HTML4 为准吧。
