Mozilla 将给 Firefox 带来私人模式

Sep 13th, 2008 | Comments

也就是传说中的和 Chrome 的差不多的 Porn Mode。

前些日子 Mozilla 的推出了个据说比 Chrome V8 还要块的 JavaScript 引擎，叫做 TraceMonkey。测试的平台比较有趣，装在 Mac mini 和 Macbook Pro 的 XP。

加上前些日子的 Mozilla Lab推出的未来浏览器的原型视频，我只能对所谓谋智公司感到失望。动作不可谓不多，然而都不怎么脚踏实地。可能是拿绿蜥蜴做图标的缘故，Fx 保持着好斗的天性。揶揄 Opera 的地址栏不如 Fx 3，最近又跟 Chrome 干上了，而 IE 就更无需多说，狐狸咬着那个对很多人来说就是 the Internet 的 IE 图标。

只可惜公司尚不允许 Chrome 使用，暂且留着这只骚动的狐狸吧。
