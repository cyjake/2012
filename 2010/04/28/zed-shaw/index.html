Zed Shaw

Apr 28th, 2010 | Comments

前一阵，proggit 出现个巨坑，原来著名文艺青年 Zed 同学喷 Dive Into Python。作为一本编程语言入门指导书，Dive Into Python 确实不咂摸好看…… 我去年开始接触 Python 的时候，看了个开头放弃了，后面基本是看一本盗版的 O’Reilly 的 Learning Python 和 Google App Engine 的官方文档。

再往前一阵，Zed 童鞋还喷过 Ruby 社区，不过事迹已经陈腐，就不再考。今天写这个主要是因为发现了他的另一个玩具网站，FretWar。fret 意思是吉他指板上的品格，就是那一条条金属条。顾名思意，该网站是用来分享社区演奏作品的。

我也弹弹吉他，所以顿时来劲，对此人颇感兴趣了。
